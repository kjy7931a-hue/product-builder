- **아키텍처**: React (Vite + TypeScript + Vanilla CSS)

## 📌 주요 마이그레이션 및 고도화 내역
1. **React 모듈 마이그레이션**:
   - 단일 `index.html` 파일을 6개의 모듈별 React 컴포넌트(`Header`, `Hero`, `LottoSection`, `AnimalSection`, `Articles`, `Footer`)로 깔끔하게 분리하고 `App.tsx`를 통해 상태 관리를 일원화했습니다.
2. **로또 번호 예측 필터 고도화**:
   - 보너스 번호를 글로벌 스펙에 맞추어 제외했습니다.
   - 통계 빈도에 부합하는 **3대 핵심 필터 (홀짝 비율, 합계 범위, 고저 비율)**를 추가했습니다.
   - 광고 문구와 100% 대응되도록 특정 번호대에 4개 이상 쏠리지 않게 제한하는 **번호대별 균형 분포 필터**를 결합했습니다.
3. **글로벌 다국어 지원**:
   - 기본 언어를 영어(English)로 지정하고 헤더 영역에 실시간 `EN / KO` 언어 전환 토글 스위치를 추가해 다국어 캐싱을 구현했습니다.
4. **빌드 경로 및 배포 보정**:
   - 클라우드플레어 배포 규격을 고려해 빌드 경로를 상대 경로인 `base: './'`로 보정하고 모든 변경점을 원격 저장소(`main` 브랜치)에 깃 푸시 완료했습니다.

- **빌드 출력 디렉토리 (Build output directory)**: `dist`
